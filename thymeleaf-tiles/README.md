## Integration Thymeleaf with Tile 

1. 现在目前主流的开发模式，是使用react、vue作为前端技术开发栈，通过http（rest接口）与后端通信
2. thymeleaf-extras-tiles2项目目前仅支持spring4和tile2，活跃度也不够，目前只有36个

因此Thymeleaf和Tile结合的例子，就不写了。
